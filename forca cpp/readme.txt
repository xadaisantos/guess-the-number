Para compilar o programa sem o Makefile, digite no terminal:
>>> g++ forca.cpp -o forca -std=c++11

Para compilar o programa com o Makefile, digite no terminal:
>>> make forca

Para rodar o programa, digite no terminal:
>>> ./forca

O arquivo palavras.txt possui a seguinte estrutura:
    Linha 1: Contem o numero de palavras no arquivo.
    Linhas subsequentes: Cada linha contem 1 palavra que sera sorteada para o jogo.

Caso queira remover uma palavra, delete a linha da palavra 
e atualize o numero de palavras na primeira linha. Caso queira
adicionar uma palavra, utilize o programa.
